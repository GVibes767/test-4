# Тест №4

Интерактивный тест по 3 уроку курса «Самопознание в свете духовных законов».

## Что внутри
- 10 вопросов
- формат ответа: Да / Нет
- пояснение после каждого ответа
- финальный экран с результатом
- призыв: «Сделайте скриншот и отправьте в наши результаты ответ.»

## Запуск
Откройте `index.html` в браузере.

## Стиль
Тёмно-синий / графитовый фон, золотистые акценты, чистая church media подача.
